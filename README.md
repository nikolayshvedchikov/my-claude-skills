# Claude Skills

Коллекция навыков и интеграций Claude с внешними сервисами.

## Подпроекты

| Проект | Описание |
|--------|----------|
| [bitrix24_restapi](./bitrix24_restapi/) | Интеграция с Bitrix24 REST API |
| [bitrix24_vibecode](./bitrix24_vibecode/) | Bitrix24 Vibe Coding скрипты |
| [avito_restapi](./avito_restapi/) | Интеграция с Avito REST API |
| [hh_restapi](./hh_restapi/) | Интеграция с HeadHunter REST API |
| [1C_restapi](./1C_restapi/) | Интеграция с 1С REST API |
| [getcourse_restapi](./getcourse_restapi/) | Интеграция с GetCourse REST API |
